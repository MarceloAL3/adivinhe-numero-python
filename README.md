# 🎯 Adivinhe o Número

Um jogo desenvolvido em Python onde o jogador tenta adivinhar um número secreto.

## 🚀 Funcionalidades

- 3 níveis de dificuldade
- Sistema de pontuação
- Histórico de tentativas
- Dicas de proximidade (Quente, Morno e Frio)
- Interface colorida no terminal

## 🛠 Tecnologias

- Python 3

## ▶ Como executar

```bash
python adivinhe_numero.py
```
